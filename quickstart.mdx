---
title: "Get started with Color Flipper: install and run locally"
description: "Clone the Color Flipper repository, install dependencies, and start the Vite dev server to have the app running at localhost:5173 in under two minutes."
---

Color Flipper runs entirely in the browser with no backend required. The only prerequisite is Node.js — once that is in place, you can have the development server running with three commands.

<Steps>
  <Step title="Install Node.js">
    Color Flipper requires [Node.js](https://nodejs.org/) to run the Vite development server and install packages. Any active LTS release works.

    Verify your installation:

    ```bash
    node --version
    ```
  </Step>

  <Step title="Clone the repository">
    Clone the Color Flipper repository and navigate into the project directory:

    ```bash
    git clone https://github.com/AbidHussainOFFICIAL/Color-Flipper.git
    cd Color-Flipper
    ```
  </Step>

  <Step title="Install dependencies">
    Install the project dependencies using your preferred package manager:

    <CodeGroup>
      ```bash npm
      npm install
      ```

      ```bash yarn
      yarn install
      ```

      ```bash pnpm
      pnpm install
      ```
    </CodeGroup>

    This installs React 19, React DOM, Vite, and the ESLint plugins listed in `package.json`.
  </Step>

  <Step title="Start the development server">
    Start the Vite dev server:

    ```bash
    npm run dev
    ```

    Vite will print a local URL once the server is ready. The output looks like this:

    ```
    VITE v7.x.x  ready in Xms

    ➜  Local:   http://localhost:5173/
    ```
  </Step>

  <Step title="Open the app in your browser">
    Open [http://localhost:5173](http://localhost:5173) in your browser. You should see the Color Flipper header and a full-bleed color preview panel.

    Click **Random** on the preview panel to generate your first color, then click the color value to copy it to your clipboard.
  </Step>
</Steps>

## Available scripts

The following scripts are defined in `package.json`:

| Command | Description |
|---|---|
| `npm run dev` | Start the Vite development server at `http://localhost:5173` |
| `npm run build` | Compile and minify the app into the `dist/` directory |
| `npm run preview` | Serve the production build locally for verification |
| `npm run lint` | Run ESLint across all source files |

<Tip>
  After the app loads, switch the **Mode** toggle to **Gradient** to generate two-color linear gradients. Gradient mode produces a `linear-gradient(colorA, colorB)` value you can paste directly into CSS.
</Tip>
