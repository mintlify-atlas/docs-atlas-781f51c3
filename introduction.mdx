Color Flipper is a modern, interactive application built with React 19 and Vite that lets you generate random colors instantly, switch between single-color and gradient modes, preview results full-bleed, and copy formatted values to your clipboard — all without a backend or external dependencies.

## What it does

Color Flipper gives you a fast, visual workflow for discovering and capturing color values:

- **Generate** — click **Random** on the preview panel to apply a new randomly-generated HEX color, or a gradient pair when in Gradient mode.
- **Preview** — the full-bleed preview panel updates in real time as colors change, so you can evaluate them at a glance.
- **Copy** — one click copies the formatted value (HEX or RGB) to your clipboard, confirmed by a brief toast notification.
- **Save and reuse** — colors are added to the History panel automatically and persisted across browser sessions via `localStorage`.
- **Tweak manually** — the Color A and Color B inputs let you type any HEX value directly without leaving the UI.
